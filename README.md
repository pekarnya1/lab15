# Домашнее задание к работе 15
## Условие задачи

6. В двумерном массиве хранится информация о зарплате 20
человек за каждый месяц года (первого человека — в первой строке, второго
— во второй и т. д.). Составить программу для расчета общей зарплаты,
полученной за год любым человеком,

## 1. Алгоритм и блок-схема
## Алгоритм
```
1) Начало.
2) Создаём двумерный массив 20×12 (люди × месяцы).
3) Заполняем массив: зарплата = (номер_человека + номер_месяца + 1) × 1000.
4) Выводим таблицу зарплат для всех людей по месяцам.
5) Запрашиваем номер человека (1-20).
6) Если номер корректен:
    Суммируем зарплату за все 12 месяцев для этого человека.
    Выводим общую зарплату за год.
  Если номер некорректен: выводим сообщение об ошибке.
7) Конец.
```

## 2. Реализация программы

```
#define _CRT_SECURE_NO_WARNINGS
#include <locale.h>
#include <stdio.h>
#include <stdlib.h>

#define PEOPLE 20
#define MONTHS 12

int main() {
    setlocale(LC_ALL, "RUS");
    int a[PEOPLE][MONTHS];
    for (int i = 0; i < PEOPLE; i++) {
        for (int j = 0; j < MONTHS; j++) {
            a[i][j] = (i + j + 1) * 1000; 
        }
    }
    printf("Таблица зарплат за год:\n");
    printf("         ");
    for (int m = 1; m <= MONTHS; m++) {
        printf("%6d", m); 
    }
    printf("\n");

    for (int i = 0; i < PEOPLE; i++) {
        printf("Человек %2d", i + 1);
        for (int j = 0; j < MONTHS; j++) {
            printf("%6d", a[i][j]);
        }
        printf("\n");
    }
    int person;
    printf("\nВведите номер человека (1-20) для расчета зарплаты за год: ");
    scanf("%d", &person);

    if (person >= 1 && person <= PEOPLE) {
        int sum_year = 0;
        for (int j = 0; j < MONTHS; j++) {
            sum_year += a[person - 1][j];
        }
        printf("Результат: общая зарплата за год = %d\n", sum_year);
    }
    else {
        printf("Ошибка: неверный номер.\n");
    }

    return 0;
}

```
